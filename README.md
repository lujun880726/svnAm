svnAm
=====

svn account&amp;path&amp;Privilege  manage svn 账号与权限管理

本工具只是为了解决不想上服务器修改配置文件，并且以前一直多人操作其他人修改配置文件不规范导致SVN不能使用查错查半天。
本工具是直接操作authz  passwd 文件进行读写，所以需要这两个文件的读写权限。

代码与结构比较糙简单用用

使用说明地址：http://blog.waiteat.com/?p=728

项目_ios
